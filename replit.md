# Royal Clean Lavanderias

## Overview
A React-based website for Royal Clean Lavanderias, a laundry service in Curitiba, Brazil. The site is built with Vite, TypeScript, and Tailwind CSS (CDN).

## Project Structure
- `/components/` - Reusable React components (Header, Footer, PromoPopup, ScrollToTop, SEOHead)
- `/pages/` - Page components (Home, About, Contact, Services, Neighborhood, NotFound)
- `/public/` - Static assets and redirects
- `App.tsx` - Main app with routing
- `index.tsx` - Entry point
- `index.html` - HTML template with Tailwind config

## Tech Stack
- React 19
- React Router 7
- Vite 6
- TypeScript 5.8
- Tailwind CSS (via CDN)
- Lucide React (icons)

## Development
- Run: `npm run dev` (port 5000)
- Build: `npm run build`
- Preview: `npm run preview`

## Deployment
- Configured for static deployment
- Build output: `dist/`
